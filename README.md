# Market Intelligence Dashboard — Setup Guide

## Project Structure

```
dashboard/
├── index.html          ← HTML5 dashboard (open in any browser)
├── scrape_crypto.js    ← Claude Code: crypto iGaming job
├── scrape_sweeps.js    ← Claude Code: sweepstakes job
├── data/
│   └── brief.json      ← Auto-generated by scripts (gitignore this)
└── logs/               ← Create manually for cron output
```

---

## 1. Prerequisites

- Node.js 20+ (check: `node -v`)
- An Anthropic API key with access to `claude-opus-4-5`
- A Slack incoming webhook URL

### Install dependencies

```bash
cd dashboard
npm init -y
npm install @anthropic-ai/sdk
```

Add to `package.json`:
```json
"type": "module"
```

---

## 2. Environment Variables

Create a `.env` file (never commit this):

```env
ANTHROPIC_API_KEY=sk-ant-...
SLACK_WEBHOOK_URL=https://hooks.slack.com/services/T.../B.../...
SLACK_CRYPTO_CHANNEL=#igaming-intel
SLACK_SWEEP_CHANNEL=#sweepstakes-intel
```

Load them before running (or use `dotenv`):

```bash
export $(cat .env | xargs)
```

Or install dotenv and add `import 'dotenv/config';` at the top of each script.

---

## 3. Run Manually

```bash
# Crypto iGaming
node scrape_crypto.js

# Sweepstakes
node scrape_sweeps.js
```

Each script:
1. Calls Claude with live web search
2. Writes/merges `./data/brief.json`
3. Posts a formatted message to your Slack channel

---

## 4. View the Dashboard

Open `index.html` in a browser. It reads `./data/brief.json` automatically.

> **Note:** Some browsers block local file fetch requests. Serve it locally instead:
> ```bash
> npx serve .
> # Then open http://localhost:3000
> ```
> Or host it on any static server (Netlify, GitHub Pages, S3).

---

## 5. Automate with Cron (macOS/Linux)

```bash
crontab -e
```

Add these lines (adjust path, runs at 07:00 and 07:15 AWST = 23:00 and 23:15 UTC):

```cron
0  23 * * * cd /path/to/dashboard && export $(cat .env | xargs) && node scrape_crypto.js >> logs/crypto.log 2>&1
15 23 * * * cd /path/to/dashboard && export $(cat .env | xargs) && node scrape_sweeps.js >> logs/sweeps.log 2>&1
```

Create the logs directory first:
```bash
mkdir -p logs
```

---

## 6. Slack Channels

Create two channels (or use one):
- `#igaming-intel` — crypto iGaming daily brief
- `#sweepstakes-intel` — sweepstakes daily brief

Add the webhook to both, or create two separate webhooks for each channel and set `SLACK_WEBHOOK_URL` accordingly per script.

## 7. Data Format (brief.json)

```json
{
  "generatedAt": "2025-04-22T23:00:00Z",
  "crypto": {
    "headline": "...",
    "summary": "...",
    "stories": [ { "title", "category", "impact", "customer", "body", "businessImpact", "source" } ],
    "watchlist": ["..."]
  },
  "sweepstakes": {
    "headline": "...",
    "summary": "...",
    "stories": [ ... ],
    "keyPlayers": [ { "name", "revenue", "trend" } ],
    "watchlist": ["..."]
  }
}
```

---

## Notes

- The two scripts are independent — run them separately or together
- They merge into a single `brief.json` so the dashboard always shows both sectors
- The dashboard falls back to demo data if `brief.json` is missing
- Add `data/brief.json` and `logs/` to `.gitignore`
